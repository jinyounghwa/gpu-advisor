## 🎯 Project Overview

**GPU Advisor** is an AlphaZero/MuZero-style planning-based AI agent that determines optimal GPU purchase timing using:
- **256-dimensional market state representation** from price, sentiment, and technical indicators
- **Monte Carlo Tree Search** (MCTS) with 50-scenario simulations
- **Neural networks** for value/policy prediction (h, g, f, a networks)
- **Automated daily data collection** from Danawa prices, exchange rates, and news

See `wiki/overview.md` for full context.

## 🛠️ Development Guidelines

### Code Changes that Require Wiki Updates

When modifying these areas, **also update the corresponding wiki page**:

| Code Change | Wiki Page | Update Rule |
|-------------|-----------|------------|
| Feature dimensions or engineering logic | `wiki/concepts/feature_engineering.md` | Update the 256D dimension table |
| MCTS parameters (simulations, PUCT, etc.) | `wiki/concepts/mcts.md` | Update the "GPU Advisor 설정" table |
| Network architecture (h/g/f/a) | `wiki/concepts/alphazero.md` | Update architecture diagram / parameter counts |
| Crawler logic or data sources | `wiki/sources/*.md` | Update frequency, fields, error handling |

### Before Implementing

1. **Read the wiki page** for the component you're modifying
2. **Check code-wiki-map.json** for all related wiki references
3. **Cross-reference** `wiki/overview.md` for system-wide context

### Code Style

- Use type hints (Python 3.10+)
- Docstrings reference wiki pages: `# See wiki/concepts/mcts.md for algorithm details`
- Keep comments minimal — detailed explanations go in wiki
- Link to wiki in complex logic: `# PUCT calculation (wiki::concepts::mcts)`
